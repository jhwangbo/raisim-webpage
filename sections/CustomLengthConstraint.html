CustomLengthConstraint

CustomLengthConstraint allows users to apply custom constraints. Users set the tension between two points using setTension().

An example is shown below:

auto wire7 = world.addCustomWire(pin7, 0, {0,0,0}, anymalB, 0, {0., 0, 0}, 2.0);
wire7->setTension(400);
