License¶

How does it work?¶

You need an activation key to use RaiSim. You can request the activation key using the RaiSim License Request Form

Go to the RaiSim License Request Form

After you submit the form, we will review your request and send you an activation key. Our review can take up to 2 days.

Please check the included examples to see how to use the activation key. The Installation section provides you with further information. Send an email to info.raisim@gmail.com for any inquiries.

Following license types are available

  • Commercial license (SLA): 1000 USD per person per year

  • Academic license (SLA): free

  • Trial license (SLA): free but only for one month
